1. What MorScan is

MorScan indexes public blockchain data from the Morpheus network on Base L2 and serves it through a dashboard and a REST API. The underlying facts (providers, bids, sessions, token balances, staking) live on a public blockchain; MorScan adds real-time indexing, derived aggregates, and cryptographically signed responses.

7. Payments

The explorer and its dashboards are free to use. Metered API access can be paid for with x402 payments: USDC on Base, either per call or as a prepaid call pack.

  • Per call: the price is shown in the 402 payment envelope the API returns (0.01 USDC per call today; the envelope is authoritative).
  • Prepaid call packs: purchased via POST /mor/v1/keys/purchase, debited per call.
  • Payments are signed transfer authorizations. We verify them at request time; on-chain settlement may happen after the request completes.
  • All payments are final. There are no refunds, including for unused prepaid calls.
